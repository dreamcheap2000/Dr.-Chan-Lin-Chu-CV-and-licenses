# TW Core IG Compliance

## Overview

PHCEP targets conformance with the **Taiwan Core Implementation Guide (TW Core IG)**
published by the Taiwan Ministry of Health and Welfare (MOHW).

Reference: https://twcore.mohw.gov.tw/ig/twcore/

FHIR version: R4 (4.0.1)

---

## De-identification FHIR Tags

All resources generated by PHCEP carry the following security tag:

```json
"meta": {
  "security": [{
    "system": "http://terminology.hl7.org/CodeSystem/v3-Confidentiality",
    "code": "R",
    "display": "Restricted"
  }]
}
```

And a data-absent extension on direct identifiers:
```json
"_name": {
  "extension": [{
    "url": "http://hl7.org/fhir/StructureDefinition/data-absent-reason",
    "valueCode": "masked"
  }]
}
```

## ICD-10-CM Coding

ICD-10-CM codes are stored in:
- `HealthObservation.icd10Code` — individual observation
- `EbmEntry.icd10Codes` — EBM statement applicability
- `MedicalQuery.icd10Code` — query classification

Coding system URI: `http://hl7.org/fhir/sid/icd-10-cm`

---

## Telemedicine Compliance

Per the Taiwan Telemedicine Act (遠距醫療法), encounters conducted via the platform
must be represented as FHIR `Encounter` resources with:
- `class` = `VR` (virtual)
- `serviceType` coding from the NHI service catalogue
- Practitioner reference with TW Core Practitioner profile

---

## Long-Term / Home Care

Care plans for home care patients use FHIR `CarePlan` with:
- `category` = `http://snomed.info/sct|736271009` (home care plan)
- `activity` items for nursing visits, medication schedules, physiotherapy
- `careTeam` referencing `TWCorePractitioner` and family caregivers
